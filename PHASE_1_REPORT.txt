KEY FINDINGS:

1. TARGET DISTRIBUTION:
   - Moderate class imbalance (67.3% vs 32.7%)
   - Will require attention during model training
   - Stratified sampling used to maintain balance in train-test

2. FEATURE CORRELATIONS:
   Top Positive Correlations with Credit Risk:
   - num_dependents: 0.039
   - age: 0.035
   
   Top Negative Correlations with Credit Risk:
   - duration_months: -0.115
   - residence_since: -0.046
   - installment_rate: -0.038

   Note: Weak correlations suggest non-linear relationships, 
   supporting use of ensemble and deep learning methods

3. AGE ANALYSIS:
   - Older customers tend to have slightly better credit
   - Wide age range provides good diversity

4. CREDIT AMOUNT:
   - Normal distribution around mean of ~10,000
   - No extreme outliers detected
   - Reasonable range for simulated lending data

5. DURATION VS AMOUNT:
   - Negative correlation with credit risk
   - Longer duration loans associated with higher default risk
   - Important feature for modeling

6. CATEGORICAL FEATURES:
   - checking_status: Strong predictor (higher balance = better credit)
   - credit_history: Critical feature (past behavior predicts future)
   - employment: Stable employment correlates with good credit
   - savings_status: Higher savings = lower risk

4. DATA PREPROCESSING STEPS
================================================================================

PREPROCESSING PIPELINE:

1. Feature Separation:
   - Numerical features identified: 7 features
   - Categorical features identified: 13 features
   - Target variable separated

2. Feature Scaling:
   - StandardScaler applied to numerical features
   - Formula: z = (x - μ) / σ
   - Scaler object saved for later use (scaler.pkl)
   - Categorical features kept as-is (already encoded 0-N)

3. Train-Test Split:
   - Split Ratio: 80% train, 20% test
   - Stratification: Applied on target variable
   - Random State: 42 (for reproducibility)
   
   Results:
   - Training Set: 800 samples
     * Good Credit: 538 (67.2%)
     * Bad Credit: 262 (32.8%)
   
   - Test Set: 200 samples
     * Good Credit: 135 (67.5%)
     * Bad Credit: 65 (32.5%)
   
   ✓ Class balance maintained across splits
